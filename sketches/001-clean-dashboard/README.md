## Variant: Clean Dashboard

### Design stance
Keep 的简洁专业 + 暗黑科技底色 + 微妙霓虹点缀。像一款高端数据仪表盘，但不冰冷。

### Key choices
- Layout: 纵向滚动，主卡片 + 横向滑动任务
- Typography: SF Pro Display / PingFang SC，干净现代
- Color: 深蓝黑底 (#0f1117) + 青色 (#00d4ff) + 紫色 (#7b61ff) 渐变
- Interaction: 圆形进度环、渐变标题、悬浮边框反馈

### Trade-offs
- Strong at: 数据一目了然，信息层级清晰，适合每天看
- Weak at: 游戏感较含蓄，不太炸裂

### Best for
- 偏理性的用户，喜欢数据驱动、不花哨的体验
