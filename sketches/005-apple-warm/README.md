## Variant: Apple Ring

### Design stance
借鉴 Apple Watch 健身圆环的视觉语言：大圆环进度作为页面中心，白底极简，数据用纯文字+emoji表达，几乎零装饰。

### Key choices
- Hero: 180px SVG圆环动画，"5周"居中
- 四列数据统计行（体重/体脂/肌肉/训练天数）
- 任务列表用极简分隔线风格（像iOS设置页）
- 快捷入口用2列网格，浅灰底色

### Trade-offs
- Strong at: 苹果生态用户零学习成本，圆环动画=成就感
- Weak at: 偏"轻"，力量感不如卡片式

### Best for
- Apple Watch用户、喜欢圆环进度隐喻的人
