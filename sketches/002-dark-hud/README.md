## Variant: Dark HUD

### Design stance
赛博朋克战术面板。全息HUD视觉、网格底纹、扫描线动画、等宽字体。用户像是一名接受任务的"特工"。

### Key choices
- Layout: 模块化面板 + 网格统计 + 任务卡片列表
- Typography: SF Mono / JetBrains Mono 等宽字体，终端感
- Color: 极暗底 (#080c14) + 纯青色 + 绿色数据 + 琥珀警告
- Interaction: 扫描线动画、边框发光、脉冲指示器

### Trade-offs
- Strong at: 科技感拉满，拍照评估特别炸，差异化极强
- Weak at: 略微"冷"，可能劝退非游戏玩家；等宽字体可读性稍弱

### Best for
- 游戏玩家、科技爱好者、想要完全不同于Keep的独特体验
