## Variant: Mission Board

### Design stance
介于 A 和 B 之间：暗色底 + 温暖紫色系 + 游戏任务卡系统。像一款精品手游的任务看板，但内容全是健身数据。

### Key choices
- Layout: 英雄横幅 + 状态圆球 + 任务等级卡 + 快捷入口
- Typography: SF Pro Display / PingFang SC
- Color: 紫色 (#7b61ff) 为主色调，金色XP，青色数据
- Interaction: XP经验条动画、任务等级标签(S/A/B/C)、浮动打卡按钮

### Trade-offs
- Strong at: 游戏感清晰（排名、XP、等级），数据不冰冷，紫色温暖
- Weak at: 任务等级标签偏游戏化，严肃健身者可能觉得不专业

### Best for
- 被游戏化激励的用户，喜欢升级/收集/成就系统
